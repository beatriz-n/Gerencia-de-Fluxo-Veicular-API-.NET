# Gerencia-de-Fluxo-Veicular-API-.NET
API desenvolvida em .NET para gerenciar o fluxo de veículos.

API desenvolvida em .NET para gerenciar o fluxo de veículos. O projeto se integra a um sistema de visão computacional, que é responsável por identificar caracteres de placas de veículos e controlar o acesso a um local específico.

A API é o back-end que gerencia os dados de acesso e veículos. O sistema de visão computacional, por sua vez, interage com a API, utilizando requisições POST para cadastrar novos dados e GET para consultar informações.
